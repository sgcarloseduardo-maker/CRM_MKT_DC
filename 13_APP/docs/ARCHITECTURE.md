# Arquitetura Inicial MVP

- App Router com módulos por domínio
- Prisma para persistência local (SQLite)
- Evolução prevista para PostgreSQL sem troca de modelo
- Single-user first, com trilha para multi-user futura

## Módulos iniciais
- Dashboard
- Marca
- Produtos Afiliados
- Prompts IA
- Campanhas
- Canais (Instagram detalhado)
