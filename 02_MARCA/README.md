# Organização de Marca

Esta pasta centraliza os ativos oficiais da marca.

- `01_logos/01_principal`: logos principais para uso institucional
- `01_logos/02_avatar`: avatars e marca reduzida
- `01_logos/03_fundo_escuro`: versões para fundos escuros
- `01_logos/04_preto_branco`: versões monocromáticas e sem fundo
- `01_logos/05_favicon`: ícones/favicons e folhas de origem
- `02_paleta`: paletas e referências cromáticas
- `03_guias`: guias oficiais (md/pdf/docx/png)
- `04_tipografia`: referências tipográficas
- `05_templates_base`: templates de uso recorrente

Assets realmente usados pelo app ficam em `13_APP/public/brand`.
